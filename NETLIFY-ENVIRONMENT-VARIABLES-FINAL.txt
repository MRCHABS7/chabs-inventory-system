Copy these environment variables to Netlify:

Variable Name: NEXT_PUBLIC_APP_NAME
Value: CHABS

Variable Name: NEXT_PUBLIC_VERSION
Value: 2.0.0

Variable Name: NEXT_PUBLIC_ENVIRONMENT
Value: production

Variable Name: NEXT_PUBLIC_STORAGE_MODE
Value: hybrid

Variable Name: NEXT_PUBLIC_ENABLE_SYNC
Value: true

Variable Name: NEXT_PUBLIC_SYNC_INTERVAL
Value: 30000

Variable Name: NEXT_PUBLIC_SUPABASE_URL
Value: [YOUR ACTUAL SUPABASE URL - e.g., https://xxxxx.supabase.co]

Variable Name: NEXT_PUBLIC_SUPABASE_ANON_KEY
Value: [YOUR ACTUAL SUPABASE ANON KEY - the long string starting with eyJhbGc...]

Variable Name: NODE_ENV
Value: production
