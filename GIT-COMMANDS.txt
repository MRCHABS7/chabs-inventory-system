Git Commands to Run (after installing Git):

1. Initialize repository:
   git init

2. Add all files:
   git add .

3. Make first commit:
   git commit -m "Initial commit - CHABS Inventory System"

4. Add GitHub remote (replace YOUR_USERNAME):
   git remote add origin https://github.com/YOUR_USERNAME/chabs-inventory.git

5. Push to GitHub:
   git push -u origin main

Note: You'll need to create the repository on GitHub first at github.com
